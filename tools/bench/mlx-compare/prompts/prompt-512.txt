The unified memory architecture on Apple Silicon lets the CPU and GPU share the same physical memory without explicit copies, which changes how a local inference runtime should schedule quantized matrix multiplications during decode. A memory-bandwidth-bound decoder reads every weight once per token, so the dominant cost is where the weights live rather than raw arithmetic throughput. Keeping eight-bit weights resident avoids streaming them from disk, and a clone-free projection path avoids copying multi-megabyte tensors on every step. Carefully measure prefill separately from decode, because prefill is compute-bound while decode is bandwidth-bound.
